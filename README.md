# Multiplayer Chess Game

A Multiplayer Chess Game.

## How to Run

```
dotnet restore
cd src/MultiplayerChessGame.Server
dotnet run
cd ../MultiplayerChessGame.Client
dotnet run
dotnet run
```

## How to Play

- Change chess piece
    - `q`, `w`, `e`, `r`, `t`, `y`, `a`, `s`, `d`, `f`, `g`, `h`
- Move selection
    - arrow keys
- Move chess
    - `z` to select source position
    - `x` to select destination
- Change side
    - `0`
- Undo
    - back
